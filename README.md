# Kanban-webserver-with-PM

## install

```bash

    wget https://github.com/jrd3n/Kanban-webserver-with-PM/archive/refs/heads/master.zip
    unzip ./master.zip
    cd ./Kanban-webserver-with-PM-master/
    python3 -m venv ./venv
    mkdir ./data
    source ./venv/bin/activate
    pip install -r ./requirments.txt
    python3 app.py

```
